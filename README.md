# Tyxel
Pyxel->Tiled Converter. Observes changes in *.pyxel file and (re-)generates a Tiled-tilesheet.  
Built ontop of https://github.com/Ragath/PyxelParser.Net/ Sample project included in Sample.json and Sample.pyxel.
